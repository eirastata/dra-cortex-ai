# DRA Cortex AI

Machine Learning and Artificial Intelligence project for data analysis, processing, and automation, with integration to services and APIs.

This project was developed as part of my practical learning journey in Data Engineering, Machine Learning, and Applied AI.

---

## Purpose

The main goals of DRA Cortex AI are:

- Automate data processing  
- Train Machine Learning models  
- Integrate with external APIs and services  
- Generate insights from data  
- Serve as a foundation for future AI projects  

---

## Technologies

- Python  
- Scikit-learn  
- Pandas  
- NumPy  
- Azure Event Hub  
- REST APIs  
- Git & GitHub
